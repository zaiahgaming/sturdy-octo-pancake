## Neon Survivor
I added a game to this project called Neon Survivor.

Run locally via python by running:
`python -m http.server 8000`

Then visit `http://localhost:8000` in your web browser.

Use WASD to move around the game. Use the mouse to point your reticle and click to shoot. Have fun!
