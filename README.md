Number Guessing Game
======================
A simple number guessing game written in Python. The game allows you to play multiple rounds, tracks your score, and has a limited number of attempts to guess the correct number. The game also keeps track of the high score, which is saved across multiple runs.

## Game Features
- Play multiple rounds
- Limited attempts to guess the correct number
- Tracks score and high score

## How to Play
1. Run the game.py file
2. Guess a number between 1 and 10
3. Try to guess the correct number within the limited attempts
4. Play multiple rounds to increase your score
5. View your current score and high score at the end of the game
