## 2024-05-18 - Optimize Enemy Movement Trigonometry
**Learning:** In hot paths like 60fps update loops running for multiple entities, calculating directional movement with trigonometric functions (`Math.atan2`, `Math.cos`, `Math.sin`) incurs significant overhead.
**Action:** Replaced trigonometric functions in the `Enemy` movement logic with basic vector normalization (`dx / dist`, `dy / dist` where `dist = Math.sqrt(dx*dx + dy*dy)`) to improve performance.
